## [8 pts] Exercise 2.
In this class we will be <i>writing</i> lots of mathematics. As such, we will be learning about \\(\LaTeX\\) (pronounced LAY-teck or LAH-teck) which is the most common way for professionals to typeset mathematics.[^1]

We write our math in code, save them as `.tex` files, and compile them to get our final document (typically a PDF).

- Download <a href="/assets/minimal_template.tex">this minimal \\(\LaTeX\\) homework template</a>.
- Go to [Overleaf](https://www.overleaf.com/project),[^2] make an account if necessary and start a new project.
- Click on the "upload" button in the upper left corner of the screen, and upload the `.tex` file you downloaded. The upload button looks like this: <img src="/assets/upload_button.png" alt="upload button" class="inline-img"/>.
- Edit the `\title` of the document to "HW 0" and the `\author` to your name.
- Under `\maketitle`, write `\section*{Exercise 2}` (because that's the one you're working on!) and on the next line, write one brief sentence about what went well or what went not-so-well with this exercise.

## [3 pts] Exercise 3.
We're going to learn how to typeset math over the course of this term, but to start to get a sense of how it works, put the following in the body of the document and compile it.

- Below the sentence you wrote, type `\section*{Exercise 3}`.

- Look at the \\(\LaTeX\) code below, and try to <u>guess</u> what it's going to look like when it compiles.

- Type (don't copy/paste) the following code in the section you just made. (You may need to scroll to the right to see the end of the code.)

```tex
The following integral converges: \[
  \int_{x=\varepsilon}^\infty \frac{1}{x^2} dx = \frac{1}{\varepsilon}.
\]
```

- Did the result look exactly how you expected? Say how it was the same or different.

- Look at the following code and <u>guess</u> what you think it's going to do. Type (don't copy/paste) the following code in the section you just made.

```tex
The following integral diverges: \(\int_1^\infty \frac 1 x dx.\)
```
